## Installation

`(recommend)` Install [Yarn](https://yarnpkg.com/) instead of NPM

```sh
$ yarn config set registry http://npm.wolaidai.com:8080/
$ yarn install
```

## Quick Start

run the app:

```bash
$ yarn run server
```

build the app:

```bash
$ yarn run build
```

Start your React.js app at [http://localhost:3000/jddv3/](http://localhost:3000/jddv3/)
